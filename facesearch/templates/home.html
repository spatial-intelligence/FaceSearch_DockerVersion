Setup

The image files to be scanned need to be placed on the host PC in the 'mapped' folder.
This 'mapped folder' will need to have been created as part of the Docker setup procedure

The structure within that 'mapped' folder should be as follows:
/search -- images to be scanned need to be placed in subfolder called 'search' (lowercase)
/target -- images which contain target faces to find
Note: Images need to be of type: .jpg or .png

When complete click [check files] to ensure the application can find the images you wish to scan.
There are two encoding methods, the HOG method is faster while the CNN method is better with a wider range of face angles (eg not looking directly at the camera) but slower.
